# Proyecto Desarrollo Web - La Galeria de Arte

Este proyecto es una plataforma web interactiva dedicada a la exploración y exhibición de diferentes estilos visuales artísticos a través de la historia. El objetivo principal es ofrecer una experiencia educativa y visualmente atractiva, permitiendo a los usuarios conocer y comparar estilos como el arte aero, cómic, cyberpunk, ochentas, graffiti, Grecia clásica, impresionismo, pixel art, retrofuturismo y ukiyo-e, entre otros.

## Estructura del Proyecto

- **index.html**: Página principal del sitio.
- **content/**: Contiene las páginas individuales de cada estilo artístico y secciones extra como juegos e información adicional.
- **assets/**: Incluye fuentes y recursos gráficos organizados por estilo.
- **img/**: Imágenes clasificadas por cada corriente artística.
- **js/**: Carpeta destinada a los scripts JavaScript del proyecto.
- **styles/**: Hojas de estilo CSS globales, compartidas y específicas para cada página o sección.

## Características principales

- Navegación temática por estilos artísticos.
- Uso de fuentes y recursos visuales personalizados para cada corriente.
- Diseño modular y escalable, facilitando la adición de nuevos estilos o secciones.
- Enfoque educativo y visual, ideal para estudiantes, docentes y entusiastas del arte y el diseño.
